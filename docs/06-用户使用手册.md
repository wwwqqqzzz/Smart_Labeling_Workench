## 系统简介

**智能打标便捷器**是一款专为货运对话数据标注设计的智能化工具，通过AI辅助和高效交互，帮助审核人员快速完成对话数据的标签审核工作。

### 核心功能

- 🤖 **AI智能推荐**: 基于RAG技术，自动推荐相关标签
- ⌨️ **快捷键支持**: 全键盘操作，无需鼠标
- 📊 **实时统计**: 审核进度、速度、预计时间一目了然
- 🚩 **疑难标记**: 标记疑难案例，建立知识库
- 💡 **标签提示**: 标签定义、使用说明实时展示

### 效率提升

- **传统方式**: 3-5分钟/条，3500条需要20-30小时
- **使用本系统**: 20-30秒/条，预计总时间4-6小时
- **效率提升**: **5-10倍**

## 快速开始

### 1. 访问系统

打开浏览器，访问: **http://localhost:3002**

### 2. 界面布局

系统采用**左右分栏**布局：

```
┌─────────────────────────────────────────────────────┐
│  左侧：对话展示区（60%）    │   右侧：操作区（40%）  │
│  ┌─────────────────────┐  │  ┌──────────────────┐ │
│  │   司机：你好        │  │  │ AI推荐标签       │ │
│  │   货主：在的        │  │  │ □ 尾板车         │ │
│  │   司机：多少钱      │  │  │ □ 冷链车         │ │
│  │   ...               │  │  │ □ 短途运输       │ │
│  └─────────────────────┘  │  │                  │ │
│                            │  │ 已选标签         │ │
│                            │  │ ✓ 钢材运输       │ │
│                            │  │ ✓ 半挂车         │ │
│                            │  │                  │ │
│                            │  │ [提交] [跳过]    │ │
│                            │  └──────────────────┘ │
└─────────────────────────────────────────────────────┘
```

### 3. 基本流程

1. **查看对话**: 阅读左侧展示的对话内容
2. **选择标签**:
   - 点击AI推荐标签（快速）
   - 或在搜索框中输入标签名称
3. **提交审核**: 点击"提交审核"按钮（或按空格键）
4. **进入下一条**: 系统自动加载下一条待审核对话

### 左侧：对话展示区

#### 对话气泡

- **绿色气泡（右侧）**: 司机说的话
- **白色气泡（左侧）**: 货主说的话

- 货物类型：`[钢材]`、`[水果]`、`[建材]`
- 运输方式：`[短途]`、`[长途]`、`[冷链]`

#### 疑难案例标记器

位于对话区下方，用于标记疑难案例：

- **标记疑难**: 点击按钮标记为疑难案例
- **添加备注**: 可以填写疑难原因（可选）
- **取消标记**: 可以取消疑难标记

---

### 右侧：操作区

#### 1. AI推荐标签

系统自动推荐3-6个最相关的标签，按相关度排序。

**操作**:
- 点击标签即可选中
- 再次点击取消选中
- 支持数字键1-6快速选择

**标签提示**:
鼠标悬停在标签上，可以看到：
- 标签定义
- 使用说明
- 示例场景

#### 2. 已选标签

显示当前已选择的所有标签。

**操作**:
- 点击标签后的 ❌ 可以取消选择
- 标签数量不限，但建议3-6个

#### 3. 操作按钮

**提交审核** (绿色按钮)
- 快捷键：`空格键`
- 保存标签并标记为"已通过"
- 自动进入下一条

**跳过对话** (灰色按钮)
- 快捷键：`Enter键`
- 跳过当前对话，不标记
- 自动进入下一条

**刷新推荐** (蓝色按钮)
- 快捷键：`R键`
- 重新获取AI推荐标签

## 功能详解

### 1. AI智能推荐

系统基于**RAG（检索增强生成）**技术，从历史标注数据中找到相似对话，推荐标签。

**推荐原理**:
1. 将当前对话转换为向量表示
2. 在向量数据库中搜索最相似的对话
3. 提取相似对话的标签作为推荐

**使用建议**:
- ✅ 优先使用AI推荐标签（准确率85%+）
- ✅ 根据对话内容适当增减标签
- ✅ 对于不明确的对话，可以使用"刷新推荐"

---

### 2. 标签搜索

如果推荐的标签不合适，可以手动搜索标签。

**操作**:
1. 点击"搜索或添加标签"输入框
2. 输入标签名称（支持拼音、简拼）
3. 选择匹配的标签

**搜索示例**:
- 输入 `lc` → 匹配 "冷链车"
- 输入 `ban` → 匹配 "半挂车"、"板材"
- 输入 `tu` → 匹配 "途"、"运输"

---

### 3. 标签分类

系统包含**23个标准化标签**，分为6大类：

#### 车辆类型
- **尾板车**: 带有尾板（升降平台）的货车
- **4.2米厢式车**: 车长4.2米的厢式货车，适合城市配送
- **冷链车**: 带有制冷设备的货车
- **9.6米**: 车长9.6米的货车，属于中型货车
- **13米低平板**: 车长13米的低平板货车，用于大件运输
- **罐车**: 罐式货车，用于运输液体、散装水泥等
- **自卸车**: 可以自动倾卸的货车，用于砂石料运输
- **半挂车**: 半挂牵引车，载重量大，适合长途运输

#### 专用车辆
- **挂衣车**: 专门用于运输服装的货车，配有挂衣架
- **集装箱车**: 用于运输集装箱的拖车
- **气垫车**: 配有气垫悬挂系统的货车，防震效果好

#### 货物类型
- **钢材运输**: 运输钢材、螺纹钢等建筑材料
- **水果运输**: 运输水果，通常需要冷链或通风
- **建材运输**: 运输建筑材料，如砂石、水泥等
- **粮食运输**: 运输粮食作物，如玉米、小麦等

#### 运输类型
- **快递运输**: 快递包裹运输，通常按单结算

#### 冷链运输
- **冷冻食品**: 需要冷冻保存的食品，温度-18°C左右
- **药品运输**: 运输医药品，需要恒温，符合GSP标准

#### 特殊运输
- **危险品运输**: 运输易燃、易爆、有毒等危险品
- **大件运输**: 运输超宽、超高、超重的货物
- **服务类型**: 如家具运输（提供搬装服务）

#### 距离分类
- **短途**: 运输距离在200公里以内
- **长途**: 运输距离超过500公里

### 4. 疑难案例标记

当遇到以下情况时，可以标记为疑难案例：

- 标签定义不清晰，无法确定应该用什么标签
- 对话内容模糊，信息不完整
- 涉及多种货物类型，标签选择困难
- 车辆类型不明确

**标记目的**:
- 建立疑难案例知识库
- 用于后续标签体系优化
- 为AI模型训练提供样本

## 快捷键操作

### 常用快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `空格` | 提交审核 | 保存当前标签，进入下一条 |
| `Enter` | 跳过对话 | 不保存标签，进入下一条 |
| `1-6` | 选择推荐标签 | 快速选择AI推荐的第1-6个标签 |
| `R` | 刷新推荐 | 重新获取AI推荐标签 |

### 快捷键使用场景

#### 高效审核流程（推荐）

```
1. 查看对话内容
   ↓
2. 按数字键选择推荐标签（如：1、3、5）
   ↓
3. 按空格键提交
   ↓
4. 自动进入下一条
```

**速度**: 10-15秒/条

#### 完整审核流程（谨慎）

```
1. 查看对话内容
   ↓
2. 点击AI推荐标签
   ↓
3. 手动搜索补充标签
   ↓
4. 查看标签提示确认
   ↓
5. 点击"提交审核"按钮
```

**速度**: 20-30秒/条

## 常见问题

### Q1: AI推荐的标签不准确怎么办？

**答**:
1. 点击"刷新推荐"按钮（或按R键）重新推荐
2. 手动搜索合适的标签
3. 标记为疑难案例，填写备注说明

---

### Q2: 一个对话应该选择几个标签？

**答**:
- **推荐**: 3-6个标签
- **最少**: 1个标签
- **最多**: 不限，但建议不超过8个

**原则**:
- 覆盖主要特征（车辆类型、货物类型、运输距离）
- 不要选择无关标签
- 避免重复含义的标签

---

### Q3: 快捷键不生效？

**答**:
1. 确保焦点不在输入框中（点击页面空白处）
2. 检查是否开启了输入法
3. 刷新页面重试

---

### Q4: 如何撤销已提交的标签？

**答**:
目前不支持撤销已提交的标签。如需修改，请联系管理员。

---

### Q5: 提交后发现标签选错了怎么办？

**答**:
1. 标记为疑难案例，备注说明选错的原因
2. 继续审核下一条
3. 定期反馈问题标签，帮助优化系统

---

### Q6: 对话内容看不懂怎么办？

**答**:
1. 标记为疑难案例
2. 备注说明"对话内容模糊"或"信息不完整"
3. 选择"跳过对话"

## 最佳实践

### 1. 审核技巧

#### 优先关注关键信息

阅读对话时，优先关注以下信息：
- **车辆类型**: 尾板车、冷链车、半挂车等
- **货物类型**: 钢材、水果、建材等
- **运输距离**: 短途、长途
- **特殊要求**: 危险品、大件、温控等

#### 合理使用AI推荐

- ✅ 大多数情况下，AI推荐准确率较高
- ✅ 对于简单对话，直接使用推荐标签
- ⚠️ 对于复杂对话，要人工复核
- ⚠️ 对于特殊对话（如危险品），要仔细确认

---

### 2. 标签选择原则

#### 完整性

覆盖对话的主要特征：
```
✅ 好的标签示例：
["冷链车", "水果运输", "短途"]

❌ 不好的标签示例：
["冷链车"]  # 缺少货物类型和距离
```

#### 准确性

选择标签要准确反映对话内容：
```
✅ 正确：
对话提到"13米半挂车"，选择标签"半挂车"

❌ 错误：
对话提到"4.2米厢式车"，选择标签"半挂车"
```

#### 简洁性

避免选择过多标签或重复标签：
```
✅ 好的示例：
["冷链车", "冷冻食品"]

❌ 不好的示例：
["冷链车", "冷冻食品", "温控-18度", "温度监控", "长途", "高速运输"]
# 标签过多，部分信息重复
```

---

### 3. 速度优化建议

#### 使用快捷键

- **纯快捷键**: 10-15秒/条（推荐熟练用户使用）
- **快捷键+鼠标**: 20-25秒/条
- **纯鼠标**: 30-40秒/条

#### 减少思考时间

- 熟悉常见标签定义
- 参考AI推荐，不要过度纠结
- 不确定时标记疑难案例，不要浪费时间

#### 批量处理

建议一次性审核20-30条对话，中间休息5分钟：
- 20条 × 20秒 = 400秒 ≈ 7分钟
- 休息5分钟
- 总计：12分钟/批次

---

### 4. 质量控制

#### 定期自查

每审核50条对话，回看前10条，检查：
- 标签是否准确
- 是否遗漏重要信息
- 是否选择了无关标签

#### 疑难案例标记

遇到不确定的情况，及时标记疑难案例：
- 帮助优化标签体系
- 为AI训练提供样本
- 建立团队知识库
